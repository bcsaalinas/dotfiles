Personal Hyprland setup on Fedora Linux

## Stack 
- **WM:** Hyprland
- **Terminal:** Kitty
- **Bar:** Waybar
- **Launcher:** Wofi
- **Notifications:** Mako
- **Wallpaper:** Hyprpaper
- **Prompt:** chromatic

Simple dark themed with pink accents, every new terminal opens with a random shiny pokemon as ASCII art, and fasfetch system info
